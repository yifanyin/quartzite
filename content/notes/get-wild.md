# Build the built environment in a less intrusive way
## [鋼構攔砂壩](https://e-info.org.tw/node/223936)
- > 穿透式的鋼構壩，在不妨礙生態系運作的情況下，達到避災減災的目的。「降雨將上游的土砂往下游帶，達到補充海岸泥沙，原是自然運作的法則。」許文奕說，而穿透式的壩體，就可讓顆粒較小、沖刷力道相對小的砂石往下游帶。
